# e-com

E-com api would a web api that would have user authentication , product, product cart ,product inventory,and an order systems.

## Features

- Signup and login
- Product
- shopping cart
- order
  
## Tech Used

- Django (Resframework)
- dj-rest-auth
- django-cors-headers
- django-extensions
- django-countries
- djangorestframework
- djangorestframework-simplejwt
- drf-yasg
- Pillow
- python-dotenv
